# Smart-City-Route-Planner-

A Python-based GUI application that finds the shortest path between cities using Dijkstra Algorithm. This project demonstrates the practical implementation of graph algorithms with an interactive user interface.

---

## Features

-  Find shortest path between cities  
-  Uses Dijkstra Algorithm  
-  Interactive GUI using Tkinter  
-  Displays path, total distance, and stops  
-  Error handling for invalid input  

---

## Technologies Used

- Python 3.x  
- Tkinter (GUI)  
- Dijkstra Algorithm  

---

## Input

- List of cities (comma separated)  
- Distance matrix  
- Source city  
- Destination city  

---

## Output

- Shortest path  
- Total distance (in km)  
- Number of stops
